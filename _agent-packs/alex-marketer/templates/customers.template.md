## 📖 Как работает этот файл

Это **клиентская реальность проекта**. Основной источник правды о том кто покупает, что болит, чего хотят, какие возражения выносят, насколько силён оффер.

**После каждой встречи** (диагностика / продающий созвон / follow-up) агент записывает карточку по шаблону ниже. Обязательны **дословные цитаты** клиента — перефразировки убивают правду.

**Каждая карточка связана:**
- 🎙 Линк на транскрипцию встречи (в `transcripts/`)
- 🧪 Линк на активные гипотезы, которые тестируются на этом клиенте (если есть)

Это **вики**: customers.md ↔ transcripts/ ↔ hypotheses.md ↔ ICP.md.
