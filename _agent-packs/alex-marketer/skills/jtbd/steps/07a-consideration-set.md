# Шаг 07a. С чем конкурируем в голове клиента

**⛔ ПРАВИЛО КОММУНИКАЦИИ: пользователь видит ТОЛЬКО результат с вопросом. НЕ видит: внутренние рассуждения, промежуточные проверки, термины JTBD. Визуальное форматирование обязательно — заголовки, выделения, не полотно текста.**

## Цель

Определить с чем продукт конкурирует в голове клиента и что конкретно не устраивает в каждом решении.

**Связь назад:** Core Job из jobStory — решения конкурируют за ПРАВО выполнить эту работу.
**Связь вперёд:** CS → Маркетинговые смыслы (скилл №5, связки «у них X → у нас Y»), Оффер (скилл №6, «почему мы»), Лендинг (скилл №10, блоки «боли» и «почему мы»). Проблемы = материал для продающих связок.

## ЧАСТЬ 1: Consideration Set

### Что СКАЗАТЬ пользователю (обязательно)

**ПЕРЕД:** > «Смотрим с чем ты конкурируешь в голове клиента. Не с другими школами — а вообще какие варианты он рассматривает.»
**НЕ ПОКАЗЫВАЙ:** термин «Consideration Set» (скажи «конкуренты в голове клиента»)

### Что входит

- Прямые конкуренты (аналогичные продукты)
- Заменители (YouTube, книги, менторы, «сам разберусь»)
- **«Ничего не делать» — ВСЕГДА в наборе!** Часто самый сильный конкурент
- Иногда: наём сотрудника, фрилансер, другая профессия вообще

### Аукцион решений

Когда работа активируется, мозг проводит внутренний аукцион. Приоритет — привычное решение. Новое побеждает ТОЛЬКО если: привычное недоступно / перестало давать ценность / новое заметно выгоднее.

### Формат

От лица клиента, связным текстом. НЕ таблица, НЕ жаргон.

```
С чем [бренд] конкурирует в голове этого клиента

У клиента [ситуация из jobStory]. Он думает «надо что-то сделать». Варианты:

«Ничего не делать» — самый сильный конкурент. Не потому что не хочет, 
а потому что страшно. Дарья: «Трусиха. Боюсь интернет-направлений.» 🟢 факт

«Курсы маркетплейсов» — WB, Ozon. Кристина: «не доучилась — сложно». 🟢 факт

«Перепродажа на Авито» — максимум 5к в месяц, не масштабируется. 🟢 факт
```

**Разметка — на КАЖДЫЙ пункт, не на блок оптом.** С подписью!

### Антипаттерн — забыть «ничего не делать»

Самая частая ошибка. «Ничего не делать» — почти всегда самый сильный конкурент. Инерция побеждает не аргументами, а отсутствием действия. Если этого нет в CS — добавь первым пунктом.

### Антипаттерн — только прямые конкуренты

❌ «Skillbox, Нетология, Geekbrains» — только прямые конкуренты
✅ «Ничего не делать + перепродажа Авито + курсы WB + SMM + вернуться в офис + прямые конкуренты» — полная картина

Клиент не выбирает между школами. Он выбирает между «вообще что-то делать или нет» и «эта профессия или другая».
