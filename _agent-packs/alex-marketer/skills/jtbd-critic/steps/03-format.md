# Шаг 03. Единообразие формата

**⛔ Покажи только найденные расхождения. Жди подтверждения.**

## Что проверяешь

- Все шапки — одинаковый формат?
- Все jobStory — структура КОГДА/ХОЧУ/ЧТОБЫ с подполями?
- Все самокритики — присутствуют у каждого сегмента?
- ВСЕ блоки у КАЖДОГО сегмента (не «аналогично сегменту 1»)
- Разметка достоверности — с подписями (🟢 факт, не голый 🟢)
- Порядок блоков одинаковый у всех сегментов

## Формат

Таблица: блок × сегмент → ✅ / ⚠️ / ❌

Покажи → жди «ок» → переходи к шагу 04.
