# Quick Start — 3 шага к первому результату

> Шаблон. Demiurg заполняет конкретными примерами из intake.

## Шаг 1: Познакомься (30 сек)
Открой Claude Code в папке офиса. Скажи:
> "Кто в команде?"

Director покажет всех агентов и их роли.

## Шаг 2: Первая задача (1 мин)
Скажи:
> "{конкретный пример из проекта пользователя}"

Director сам определит кому передать.

## Шаг 3: Приоритеты (30 сек)
Скажи:
> "Что на сегодня?"

Director покажет текущие задачи из context.md.
