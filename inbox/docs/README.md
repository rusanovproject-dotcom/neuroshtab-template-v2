# inbox/docs/ — твои исходные материалы

Сюда кидай документы о себе и проекте **до** запуска `/strategist-intake`:

- Лендинг / сайт — скриншоты или текстовая копия
- Оффер / прайс
- 3-5 лучших своих постов или статей (для voiceprint — чтобы будущий копирайтер-помощник писал твоим голосом)
- 1-2 кейса клиентов (истории успеха как ты их обычно рассказываешь)
- План года / квартала если есть
- Таблица клиентов / лидов если не жалко показать
- Бренд-ассеты (цвета, шрифты, логотип) если есть

**Форматы:** md, txt, pdf (текстовые), html, изображения.

**Главное правило:** не полируй, кидай как есть. Чем больше сырья — тем живее распаковка.

После того как закинул всё — запусти `/strategist-intake`. Стратег прочитает, классифицирует, распакует в черновики артефактов проекта.
