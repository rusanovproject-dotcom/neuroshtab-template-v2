# Инбокс

Кидай сюда всё непонятное — файлы, ссылки, заметки, скриншоты, идеи на бегу.

Потом запусти:

```
/intake
```

Агент ПАРА разберёт содержимое и разложит по правильным папкам: `projects/`, `clients/`, `knowledge/` или оставит здесь с пометкой если не хватает контекста.

**Главное правило:** лучше кинуть в инбокс и разобрать позже, чем потерять.
