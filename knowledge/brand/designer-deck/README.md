# Designer Pitch Deck

**Что это:** 12-слайдовая презентация агента Дизайнер из AI-офиса. Используется как маркетинговый ассет для презентации возможностей агента-дизайнера в твоём офисе.

**Стиль:** cinematic brutalist-editorial на графитовом фоне (#0a0a0a) с ротацией неонового акцента. Шрифты: Unbounded (display), Space Grotesk (sec display), Inter (body).

## Слайды (12)

1. **ДИЗАЙНЕР** — cover
2. **РАНЬШЕ БЫЛО ТАК** — before (как было без агента)
3. **ТЕПЕРЬ** — after (с агентом)
4. **ПАМЯТЬ БРЕНДА** — superpower 1
5. **ДЕЛАЕТ ВСЁ САМ** — superpower 2
6. **АРХИТЕКТУРА СМЫСЛА** — superpower 3
7. **ВКУС** — superpower 4
8. **САМОКРИТИКА** — superpower 5
9. **РЕФЕРЕНС-МАГИЯ** — superpower 6
10. **НЕ ТОЛЬКО ЛЕНДИНГ** — superpower 7
11. **ЖИВАЯ ПАМЯТЬ** — superpower 8
12. **ОДНА КОМАНДА** — final

## Как пользоваться

- **Открыть** в браузере: `designer-deck.html` — двойной клик
- **Навигация внутри:** стрелки ← →, PgUp/PgDn, Space, Home/End, цифровые клавиши 1-9, R для сброса
- **Сохранить в PDF:** открой `designer-deck-print.html`, Cmd+P → Save as PDF (каждый слайд = одна страница)

## Параметры (Tweaks-панель внутри)

Работают клавиши T/panel для настройки:
- `accent` — цвет неонового акцента (default `#ff2e17`)
- `paper` — цвет фона (default `#0a0a0a`)
- `intensity` — интенсивность эффектов (0-5, default 5)
- `shake` — анимация дрожания (default on)
- `caps` — caps lock для титулов

## Использование в офисе

Маркетинговый ассет для:
- Рассылок клиентам (объявление нового агента в офисе)
- Презентации программы в живых разговорах
- Встраивания в лендинг (как интерактивная секция-презентация)

## Технические детали

- HTML полностью самодостаточный — `deck-stage.js` встроен inline
- Дизайн-размер 1920×1080, автоматически масштабируется под экран
- Состояние слайда сохраняется в localStorage (по пути документа)
- Print-версия готова отдельным файлом на случай если основной не режет чисто на страницы
