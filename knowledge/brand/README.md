# Brand Books по проектам

Здесь живут Brand Book-файлы для каждого проекта клиента.

## Структура

```
knowledge/brand/
├── README.md           (этот файл)
├── {project-slug}/
│   └── brand-book.md   (визуальная конституция проекта)
├── {another-project}/
│   └── brand-book.md
```

## Как создаётся

Brand Book собирается через Дизайнера — скажи ему:
> "Собери мне Brand Book для проекта [название]"

Он проведёт 10-минутный диалог: настроение → референсы / стилевые направления → 3 варианта на выбор → финальный артефакт. Файл попадёт в `knowledge/brand/{project}/brand-book.md`.

## Что внутри Brand Book

- Палитра (4 hex-кода: фон / основной / акцент / текст)
- Шрифты (2 шрифта: заголовки + текст, с Google Fonts ссылками)
- Настроение (3-4 прилагательных)
- Запреты (что НЕ использовать)
- Do/Don't примеры
- Референсы (если были)

## Как используется

Перед любым дизайном Дизайнер читает соответствующий `brand-book.md` — оттуда берёт палитру, шрифты, настроение, и вставляет в промт для Claude Design или Claude Code.

## Как редактировать

Просто скажи Дизайнеру: *"Замени шрифт заголовков на Playfair"* или *"Добавь ещё один акцентный цвет — зелёный"*. Он обновит файл и зафиксирует изменение в истории.
