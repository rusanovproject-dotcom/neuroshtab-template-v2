# База знаний

Здесь хранятся знания, которые важно сохранить: инсайты, шаблоны, исследования, паттерны работы.

Чтобы добавить новую карточку знания, запусти:

```
/new-knowledge
```

Скилл создаст карточку и автоматически добавит запись в `INDEX.md`.

**Когда сюда класть:**
- Узнал что-то важное про нишу или аудиторию
- Сделал шаблон который работает
- Провёл исследование конкурентов
- Записал решение задачи которая может повториться
