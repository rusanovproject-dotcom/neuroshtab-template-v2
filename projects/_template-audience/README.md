## 📌 Закон Stage 1

1. **Только ЦА.** Не лезем в продукт/оффер/воронку — это другие стадии.
2. **Из данных, не из головы.** Гипотезы — из реальных цитат, отзывов, интервью. Без данных = `[ГИПОТЕЗА N%]`.
3. **Дословные цитаты > перефразировки.** Если клиент так не говорил — поле пустое.
4. **Stop+wait после каждого сегмента.** Без согласия владельца дальше не идём.
5. **Никаких пустых папок-органов.** Нет `product/`, `funnel/`, `brand/` пока не активирована соответствующая стадия.

## Как работать

### Запуск распаковки (один раз)
- **«распакуй мою ЦА»** / **«найди сегменты»** / **`/audience-stage`** → 4 фазы (A→B→C→D), на выходе ТОП-3 в NORTH-STAR.md + `voice-of-segment.md` × N

### Углубление каждого сегмента
- **«awareness для {slug}»** / **`/audience-awareness-lite {slug}`** → awareness + lingvo + segment-core.md

### Конкурентный анализ (опционально)
- **«разведай конкурентов»** / **`/competitors-research`** → `intel/competitors-{slug}/`

### Постоянная ревизия (раз в неделю)
- **«пятничная ревизия»** / **«обнови сегменты»** / **`/revise-segment`** — каждую пятницу 16:00. Переносит наблюдения из `segment-observations.md` в `segment-portrait.md` и `segment-core.md`.

### Любые входящие (всегда)
- Цитата клиента / коммент / голосовая / скрин → `inbox/_new/`
- **«разбери инбокс»** → Алекс классифицирует и раскладывает
