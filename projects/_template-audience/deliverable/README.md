# Deliverable — артефакт Phase E

Сюда складывается финальный HTML-отчёт по Stage 1 — `audience-report.html`. Собирается скиллом `/audience-deliverable` из закрытых артефактов (`NORTH-STAR.md`, `segment-portrait.md` × N, `voice-of-customer.md`, `segment-core.md` для hot).

Открывается локально из браузера, можно отправить ментору / партнёру / инвестору как один файл.

Запускается автоматически в конце `/audience-stage` после `/accept` Stage 1, или вручную через триггер «собери отчёт по ЦА».
