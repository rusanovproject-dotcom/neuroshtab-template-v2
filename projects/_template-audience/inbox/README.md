# Inbox — желудок Stage 1

Сюда падает **всё что относится к ЦА**: цитаты, скрины комментов, голосовые с инсайтами, ссылки на конкурентов, транскрипты интервью / кастдев / выступлений.

## Как кидать

1. Любой файл/мысль → `_new/{YYYY-MM-DD-HHMM-короткий-slug}.md`
2. Скажи Алексу: **«разбери инбокс»** — он классифицирует и разложит:
   - Цитаты → `audience/voc.md`
   - Транскрипты интервью / кастдев → `audience/research/transcripts/`
   - Конкуренты (URL/скрины/анализ) → `intel/competitors-{slug}/`
   - Идеи / гипотезы → `hypotheses.md → 🧊 Backlog`
3. Если непонятно куда — Алекс спросит.

## Папки

| Папка | Что |
|-------|-----|
| `_new/` | Свежак — неразобранное (лимит 20 файлов) |
| `_processed/{YYYY-W{N}}/` | Разобранное по неделям |
| `_raw/transcripts/` | Сырые транскрипты до классификации |
| `_raw/screens/` | Скриншоты комментов / чатов |
| `_raw/audio/` | Голосовые .m4a/.ogg |
| `_raw/refs/` | Ссылки/PDF/скриншоты конкурентов |

## Frontmatter (опционально)

```yaml
---
type: quote | transcript | competitor | idea | reference
tags: [ICP, SEGMENT, MESSAGE]
source: "коммент в TG" / "кастдев с {имя}" / "пост конкурента"
---
```

Если frontmatter нет — Алекс поставит сам или спросит.

## Важно

На Stage 1 принимаем **только то что про ЦА**. Если кидаешь идею про продукт или воронку — Алекс положит в Backlog с пометкой «ждёт Stage 2/3».
