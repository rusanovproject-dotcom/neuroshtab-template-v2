# Проекты

Папка проектов — каждый проект живёт в своей подпапке.

Чтобы создать новый проект, запусти:

```
/new-project
```

Скилл спросит: цель проекта, целевую аудиторию, ключевые метрики успеха — и автоматически создаст структуру папки с заполненным `CLAUDE.md`.

Посмотри `_example-project/` — там пример того, как выглядит готовый проект.

**Структура проекта:**
```
projects/
  название-проекта/
    CLAUDE.md       — контекст и цель (главный файл)
    notes/          — заметки, исследования
    artifacts/      — готовые результаты
```
