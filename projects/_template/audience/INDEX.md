# Audience — КТО

🧠 Доля мозга «кому продаём». Выводы о целевой аудитории.

## Файлы (наполняются владельцем / маркетологом)

| Файл | Что | Кто пишет |
|------|-----|-----------|
| `NORTH-STAR.md` | ICP + JTBD + Main Pain + JTBD + топ цитаты | владелец |
| `voc.md` | 👄 Voice of Customer — цитаты ЦА с источниками (append only) | маркетолог |
| `{segment-slug}/base.md` | Под-сегмент: ядро + Quotes & Vocabulary | маркетолог из паттернов |
| `{segment-slug}/stream.md` | Live-наблюдения по сегменту | маркетолог |

## Папка research/

`research/transcripts/{YYYY-MM-DD}-{slug}.md` — исследовательские материалы: касдев, опросы ЦА, выступления владельца, разборы интервью, ресёрч ниши.

Сюда падают записи **не привязанные к конкретному клиенту**. Если есть встречи с лидами — они живут в `customers/{name}/transcripts/` (через модуль встреч).

## Источник цитат для voc.md

Голос ЦА собирается из любых источников:
- Комментарии под постами и в чатах
- Отзывы и переписки
- Ответы на опросы
- Реплики из транскриптов встреч (если модуль активирован)
- Цитаты из интервью

Каждая запись с указанием источника и даты.

## Связь с тестами

Гипотезы с тегом `[ICP]` валидируют `NORTH-STAR.md`. Тег `[SEGMENT]` — обновляет конкретный под-сегмент.

Под-сегменты создаются НЕ заранее, а по факту: 3+ упоминаний или покупок одного типа → выделяется новый сегмент.
