## 📌 Закон проекта

1. **Реальность диктует выводы.** Файлы в product/audience/brand/funnel пишутся не из догадок, а из валидированных гипотез или прямой воли владельца.
2. **Агент не перезаписывает голос.** Изменения предлагает в `hypotheses.md → Log`, ты принимаешь командой `/accept H{N}`.
3. **Любой вход — через inbox.** Идея, цитата, конкурент, скрин, цифра — сначала в `inbox/_new/`, потом классификация.
4. **Гипотеза без тега и метрики — не гипотеза.** Идея в Backlog без критерия успеха не двигается в Active.

## Как работать

- **Что-то прилетело** → в `inbox/_new/`
- **«Разбери инбокс»** → агент-маркетолог классифицирует и раскладывает
- **«Проверь Active гипотезы»** → агент прогоняет дедлайны и прогресс
- **Раз в неделю** → еженедельный чек-ин + ротация `inbox/_processed/`
