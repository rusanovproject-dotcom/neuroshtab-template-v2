## Шаблон гипотезы (для копирования)

```
### H{N} [PRIMARY/secondary] — «короткое название»
- **Источник:** inbox/.../{file}.md или customers/{name}/{date}.md
- **Связь:** сегмент / клиент / канал
- **Уверенность:** 0-100%
- **Тест:** что именно делаем (объём, длительность, выборка)
- **✅ Успех:** конкретная цифра/порог
- **❌ Провал:** конкретная цифра/порог
- **Counterfactual tested:** yes/no (без yes — Key Learning не присваивается)
- **Supersedes:** H{X} (если заменяет старую validated)
- **Дедлайн:** YYYY-MM-DD
- **Прогресс:** N/M · комментарий
```

## Правило обновления файлов выводов

**Агент НЕ перезаписывает** `product/core-offer.md`, `brand/positioning.md`, `audience/NORTH-STAR.md` и т.д. напрямую. При валидации H агент **предлагает diff в Log**:

```
## YYYY-MM-DD — предложение из H{N}
Файл: product/core-offer.md
Предлагаемое изменение:
[diff]
Эксперт принимает (`/применить H{N}`) или отвергает.
```

Эксперт сам решает что переносить. Голос остаётся за экспертом.
