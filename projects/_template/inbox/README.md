# Inbox — желудок проекта

Сюда падает **всё**: идеи, цитаты ЦА, конкуренты, скрины, метрики, голосовые, транскрипты, ссылки.

## Как кидать

1. Любой файл/мысль → `_new/{YYYY-MM-DD-HHMM-короткий-slug}.md`
2. Скажи маркетологу: «**разбери инбокс**» — он классифицирует и разложит.
3. Если непонятно куда — он спросит.

## Папки

| Папка | Что |
|-------|-----|
| `_new/` | Свежак — неразобранное (лимит 20 файлов, потом hard-stop) |
| `_processed/{YYYY-W{N}}/` | Разобранное по неделям (с пометкой «→ ушло туда-то») |
| `_raw/transcripts/` | Сырые транскрипты до классификации |
| `_raw/screens/` | Скриншоты |
| `_raw/audio/` | Голосовые .m4a/.ogg |
| `_raw/refs/` | Ссылки/PDF/референсы лендингов и рассылок |

## Frontmatter (опционально)

```yaml
---
type: idea | transcript | quote | competitor | metric | case | reference
tags: [ICP, OFFER]   # 1-3 тега из словаря hypotheses.md
source: "голосовая" / "встреча с {имя}" / "пост в TG"
---
```

Если frontmatter нет — маркетолог поставит сам или спросит.

## Ротация

- Каждый понедельник создаётся новая папка `_processed/{YYYY-W{N}}/`
- `_raw/` старше 90 дней → маркетолог предложит архивировать в `../_archive/`
