# Клиенты

Папка клиентов — каждый клиент живёт в своей подпапке.

Чтобы добавить нового клиента, запусти:

```
/new-client
```

Скилл создаст подпапку с заполненной карточкой по шаблону.

Шаблон карточки клиента лежит в `_template/` — можно посмотреть как пример.

Реестр всех клиентов — в `INDEX.md` этой папки.
