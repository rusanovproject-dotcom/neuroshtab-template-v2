# Как работать с Дизайнером

Короткий гайд для тебя — как запускать дизайн в офисе.

## Два места где делается дизайн

**Claude Code — это тот чат где ты сейчас со мной разговариваешь.** Тут Дизайнер пишет тебе инструкции. Когда даю шаблон для интерактивной штуки (лендинг, дашборд, презентация) — ты копируешь его, открываешь **новый чат в Claude Code** (плюсик слева сверху) и вставляешь. Через минуту справа появляется живая страница — можно кликать, скроллить.

**Claude Design — отдельный сайт claude.ai/design.** Открой его в новой вкладке. Туда вставляешь шаблоны для картинок (обложки, карточки, постеры). Через 30 секунд получаешь картинку — можно скачать.

**Если перепутаешь** — не беда, скажи мне «куда это», я подскажу.

Если у тебя нет Claude Pro — не страшно, есть бесплатные альтернативы (Google Gemini, v0.dev, Ideogram). Скажи — подберу под твою задачу.

## Что Дизайнер умеет

- Собирает **Brand Book** твоего проекта: палитра, шрифты, настроение, запреты. Живой файл, редактируется.
- Пишет **промты для генерации дизайнов** под твой стиль — чтобы ты просто скопировал и получил результат.
- Работает с двумя стеками: **Claude Design** (картинки) и **Claude Code** (интерактивные артефакты).
- Помнит твои визуальные предпочтения от сессии к сессии.

## Что Дизайнер НЕ делает

- Не генерирует сам картинки — только пишет промт, ты копируешь и запускаешь.
- Не пишет тексты на визуалах (это копирайтинг).
- Не верстает продакшн-сайты (только прототипы через Claude Code).

## С чего начать

**Первый заход — Brand Book.** Скажи Дизайнеру:
> "Хочу собрать Brand Book для [название проекта]"

Он проведёт тебя через 5-10 минут диалога: спросит настроение, попросит референсы или предложит на выбор стилевые направления, даст 3 варианта Brand Book — выбираешь один. Готово.

Без Brand Book каждый дизайн будет в другом стиле — зоопарк. С ним — всё в единой системе.

## Как попросить конкретный дизайн

Просто скажи что нужно. Примеры:
- *"Хочу лендинг под мой курс"*
- *"Нужна обложка для Telegram-канала"*
- *"Сделаем презентацию на 10 слайдов для инвесторов"*
- *"Дашборд моих целей на неделю"*

Дизайнер определит тип, выберет стек, напишет промт и объяснит куда его вставить.

## Два пути: Claude Design vs Claude Code

**Claude Design** (claude.ai/design):
- Для **картинок**: постеры, обложки, карточки, иллюстрации.
- Быстро: вставил промт → через 30 секунд готово.
- Открывай: claude.ai/design

**Claude Code** (этот чат / артефакты):
- Для **интерактива**: лендинги, дашборды, презентации, прототипы приложений.
- Результат — HTML/React артефакт, можно кликать, скроллить, встраивать.
- Запускается прямо в Claude Code — Дизайнер даст промт, ты его пришлёшь отдельным сообщением.

Если не знаешь что нужно — просто опиши задачу словами. Дизайнер сам решит.

## Коротко

1. **Brand Book** — первый шаг. Один раз собрали → используется во всех дизайнах.
2. **Скажи задачу** — он выберет тип и стек.
3. **Скопируй промт** — вставь в нужный инструмент.
4. **Вернись с результатом** — если не нравится, докрутим.
