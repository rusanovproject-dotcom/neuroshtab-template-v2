## Цель

Оценить, стоит ли вкладываться в этот сегмент.

**Связь назад:** все предыдущие шаги этого сегмента (02-09).
**Связь вперёд:** оценка → ранжирование (шаг 11). Выбор фокусного = куда ресурсы. Оффер (скилл №6) и лендинг (скилл №10) → для фокусного.

## Что СКАЗАТЬ пользователю (обязательно)

**ПЕРЕД:** > «Оценим, стоит ли на эту группу тратить силы. Сколько таких людей, есть ли деньги, сильно ли достало.»
**ПОСЛЕ:** > «Вот оценка по 5 параметрам. Главное: [ключевой вывод]. Согласен?»
**НЕ ПОКАЗЫВАЙ:** термины TAM/SAM/SOM (скажи «сколько всего / сколько достанем / сколько реально за год»)

---

## 5 обязательных элементов (не пропускать ни один)

### 1. ✅ Размер (TAM → SAM → SOM)

- **TAM** — весь рынок людей с этой задачей
- **SAM** — сколько из них мы реально можем достать (география, каналы, язык)
- **SOM** — сколько реалистично получить за год при текущих ресурсах

**Конкретные цифры или порядок величины.** НЕ «крупный», НЕ «большой».

❌ «Размер: крупный» — не оценка
✅ «TAM: ~500-800к (женщины в декрете, ищущие удалённый заработок). SAM: 50-100к (готовы на платное обучение). SOM: 1000-1500 покупок/год при текущих каналах. 🟡 гипотеза»

Если нет данных — порядок величины (сотни / тысячи / десятки тысяч) + 🟡 гипотеза. Не выдумывай точные цифры.

### 2. ✅ Платёжеспособность

Сколько готовы платить, в каком формате (разово / рассрочка). Есть ли финансовые блокеры.

**Пример:** «Низкая-средняя. Своего дохода нет. Покупка через рассрочку 5000/мес. Отказ банков у 15-20%. 🟢 факт»

### 3. ✅ Конкуренция

Низкая / средняя / высокая — с пояснением.

**Пример:** «Средняя. Прямых конкурентов по Яндекс Директ мало, но за внимание конкурируют все удалённые профессии (WB, SMM, дизайн). 🟡 гипотеза»

**Антипаттерн:** «Конкуренция: высокая» — без пояснения. Высокая с кем? На каком уровне?

### 4. ✅ Сила проблемы (1-10)

Насколько достало текущее решение. Оценивать по трём вещам:
- Разочарование результатом (из проблем текущих решений, шаг 07)
- Негативные эмоции (оттуда же)
- Количество лишних действий / налоговых работ (оттуда же)

**Пример:** «7-8. Финансовая зависимость + потеря самоидентификации — сильный дискомфорт, но не катастрофа (муж зарабатывает). 🟢 факт»

### 5. ✅ Можем ли сделать лучше

Конкретно за счёт чего. Не «конкурентное преимущество» — а: чем текущие решения проигрывают? Что мы делаем такого, чего они не делают?

**Пример:** «Да. Структурированный курс с куратором, первый результат на 4-5 неделе, два пути (фриланс + найм). Альтернативы (самообучение, бесплатные курсы) этого не дают. 🟢 факт»

## Антипаттерн из прогонки — «крупный» вместо цифр

Пример: скилл написал «Размер: крупный» и перешёл дальше. Пришлось возвращать и просить конкретику. После правки — порядок величины с логикой расчёта (Росстат → фильтр → воронка).

**Правило:** если написал «крупный/средний/маленький» без цифр — это не оценка. Перепиши.

## Антипаттерн: оценка одним словом

❌ «Платёжеспособность: средняя» — для кого средняя? 5к/мес или 50к?
✅ «Низкая-средняя. Нет дохода, покупка через рассрочку 5000/мес. Отказ банков у 15-20%.» — с контекстом.

❌ «Сила проблемы: 8» — голая цифра.
✅ «7-8. Зависимость финансовая + потеря самоидентификации. Но муж обеспечивает — не катастрофа.» — с аргументацией.

## Как считать TAM/SAM/SOM без данных

**Шаг 1. Отправная точка:** Росстат (демография), WordStat (запросы), SimilarWeb (конкуренты), отчёты.

**Шаг 2. Фильтруй сверху вниз:**
```
Росстат: ~3-4 млн в декрете одновременно
→ 15-25% ищут заработок → 500-800 тыс. (TAM)  🟡 гипотеза
→ 10-15% готовы платить → 50-100 тыс. (SAM)   🟡 гипотеза
→ текущие каналы → 1000-1500/год (SOM)          🟡 гипотеза
```

**Шаг 3. Помечай:** каждый фильтр = гипотеза. «15-25%» — здравый смысл, не данные.

**Шаг 4. Порядок важнее точности:** не «73 482», а «десятки тысяч». Этого достаточно для решения.
