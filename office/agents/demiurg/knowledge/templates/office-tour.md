# Office Tour — интерактивная экскурсия по AI-офису

> Шаблон. Demiurg вставляет эту секцию в Director CLAUDE.md.
> Активируется когда пользователь говорит: "покажи офис", "экскурсия", "tour", "кто в команде".

## Скилл: Office Tour

Когда пользователь просит показать офис, проведи экскурсию:

### Шаг 1: Обзор (5 сек)

```
Твой AI-офис:

{N} агентов | {M} проектов | Паттерн: {pattern}
Создан: {date}
```

### Шаг 2: Команда (по одному)

Для каждого агента покажи карточку:

```
┌─────────────────────────────┐
│  {Имя агента}               │
│  Роль: {роль одним словом}  │
│  Зона: {что делает}         │
│  Скажи: "{пример фразы}"   │
└─────────────────────────────┘
```

Показывай по одному, с паузой "Дальше?" между ними.

### Шаг 3: Схема связей

```
[Ты]
  |
[Director] — принимает всё
  |
  ├── [{Агент 1}] — {зона}
  ├── [{Агент 2}] — {зона}
  └── [{Агент 3}] — {зона}
```

### Шаг 4: Попробуй

```
Готово! Попробуй прямо сейчас:

1. "{задача из основного проекта}"
2. "{задача из контента}"
3. "Что в приоритете?"
```

## Правила tour

- Не показывай пути к файлам — пользователю это не нужно
- Говори простым языком, без терминов
- Если агент 1 — не показывай "команду", просто покажи что умеет
- Если пользователь говорит "хватит" — останови tour
