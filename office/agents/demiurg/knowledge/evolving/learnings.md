# Learnings — что узнал из практики

Сюда **агент пишет сам** после каждой сборки/аудита. Append-only. Формат:

```
## YYYY-MM-DD: Заголовок открытия

**What:** что произошло
**Why:** почему это важно
**Pattern:** какое правило выводим на будущее
```

Перед сборкой — `grep` по этому файлу на ключевые слова домена. Не повторяй уроки прошлых сборок.
