# Debug Playbook — когда агент делает не то

## Диагностика

| Симптом | Причина | Фикс |
|---------|---------|------|
| Агент читает не то, забывает цель | Контекст переполнен | /clear + перезапуск |
| Агент "застревает" между двумя вариантами | Конфликтующие инструкции | Убрать одну, сделать explicit |
| Делает не то что просили | Vague instruction | Добавить: "измени X, проверь Y, результат = Z" |
| Действует по старой информации | context.md / state.md устарел | Обновить перед задачей |
| Не видит нужные файлы | .claudeignore блокирует | Проверить .claudeignore |
| Блуждает без плана | Нет Issue / acceptance criteria | Создать Issue с чёткими критериями |

## Первая помощь

```
1. Проверить контекст → сколько токенов? (при >60% → новая сессия)
2. Прочитать CLAUDE.md → есть противоречия?
3. Проверить .claudeignore → не блокирует нужное?
4. Посмотреть последние 5 сообщений → совпадают с целью?
```

## Когда что делать

**Retry с уточнением:** агент понял идею, но реализация неполная.
"Добавил 2 из 3 фич → дополни оставшиеся"

**Новая сессия:** контекст >60%, агент начал путаться.
Скопировать актуальный контекст из state.md → /clear → продолжить.

**Эскалация человеку:** агент делает логически неправильные вещи.
Удалил нужный файл, применил не ту версию, зациклился.

## "Учись читать логи"

> Фраза из Яндекса, которая решает всё.

Логи агента = единственный trust-source. Подключи трейсинг (LangFuse),
читай что агент реально делает, а не что он говорит что делает.

## Цикл самоотладки (из OpenAI Harness)

Когда агент видит своё окружение (логи, спаны, метрики):
```
Агент → делает → пишет логи → читает свои логи →
→ видит ошибку → пишет фикс → деплоит → проверяет → повтор
```

Дай агенту доступ к: логам, метрикам, коду, API — и он сможет себя чинить.

## Паттерн Memory через скиллы

Когда агент научился чему-то новому:
1. Скажи: "напиши список инструкций для себя на будущее"
2. Он генерит step-by-step
3. Положи в CLAUDE.md или отдельный скилл
4. Следующий запуск — агент уже знает маршрут

Это самообучение через файловую систему. Без ML, без дообучения.
