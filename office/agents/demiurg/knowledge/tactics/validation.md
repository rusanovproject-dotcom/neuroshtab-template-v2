# Как тестировать агентов и системы

## LLM-as-Judge протокол

После сборки системы оцени каждый файл по 5 осям (0-10):

| Ось | Что оцениваем | Как проверить |
|-----|--------------|---------------|
| **Навигация** | Агент найдёт файл за ≤2 шага | Пройди путь: задача → файл |
| **Экономия** | Нет лишнего в контексте | Identity ≤300 tok? CLAUDE.md ≤200 строк? |
| **Связность** | Все ссылки валидны | Каждый путь в handle → файл существует? |
| **Единственность** | Нет дублей | grep по ключевым фразам в разных файлах |
| **Ясность** | Понятно с первого прочтения | Позитивные формулировки? Есть примеры? |

Score = среднее по 5 осям. Цель: ≥ 8/10.

## Тестовые сценарии

Для каждого агента — 2 типичные задачи. Формат:

```
Сценарий: "Напиши пост про мастер-группу"
Путь: Director → AGENTS.md → CMO → knowledge/INDEX.md → brand/ + methodology/
Ожидание: CMO находит нужные файлы за 3 шага
Результат: pass/fail
```

### Типы сценариев

1. **Happy path** — типичная задача, всё на месте
2. **Cross-agent** — задача требует передачи между агентами
3. **Edge case** — файл в неожиданном месте, агент должен найти через INDEX

## Чеклист ошибок (Red Flags)

Перед финализацией пройди по чеклисту:

- [ ] Broken links — ссылка на файл которого нет
- [ ] Дубли — один факт в 2+ местах
- [ ] Ghost context — агент полагается на то что "помнит" из прошлой сессии
- [ ] Over-kitchen-sink — CLAUDE.md перегружен правилами "на всякий случай"
- [ ] Негации — "НЕ делай X" без объяснения почему и без позитивной альтернативы
- [ ] Пересечения — два агента отвечают за одно и то же
- [ ] Нет INDEX.md — папка с >3 файлами без навигации
- [ ] Inline контент — содержимое файла скопировано в CLAUDE.md вместо ссылки

## Self-Refine петля

```
Раунд 1: Judge → scores [7, 6, 9, 8, 7] avg=7.4
  → Найди оси < 8 → конкретные фиксы → примени

Раунд 2: Re-Judge → scores [8, 8, 9, 9, 8] avg=8.4
  → Все ≥ 8 → ВЫХОД

Правило остановки:
- avg ≥ 8 → достаточно
- delta между раундами < 0.5 → стабилизировалось
- 3 раунда max → хватит
```

## Метрики для разных ролей

| Агент | Что проверять |
|-------|--------------|
| Director | Правильно роутит 3 из 3 тестовых задач? |
| CTO | Находит код проекта за 1 шаг? Не лезет в маркетинг? |
| CMO | Находит brand/ и audience/ за 2 шага? |
| Любой | Identity ≤ 300 tok? Есть "НЕ отвечает за"? |
