## 💡 КЛЮЧЕВЫЕ КОНЦЕПЦИИ

### 3W Модель (What-Why-Who)
- **What:** Что покупает/делает потребитель?
- **Why:** Почему он это покупает? (разные люди = разные причины)
- **Who:** Кто этот человек? (конкретный портрет)

### Пирамида осознанности
- 🔥 **Горячый сегмент** (100% в него верю, высокая конверсия)
- ☀️ **Теплый сегмент** (70-80% верю, нужно убедить)
- ❄️ **Холодный сегмент** (50-60% верю, долгосрочная ставка)

### CRITICAL: Разница проблема vs потребность
- **Проблема:** Болит, не хватает (дефицит)
- **Потребность:** Хочу, мечтаю (желание решить проблему)

## 📈 СЛЕДУЮЩИЕ ШАГИ

После этого скилла:
1. **User Research** — интервью с каждым из 3 сегментов
2. **Map of Awareness** — распределение ответов по уровню осознания
3. **Контент-стратегия** — разный контент для каждого уровня
4. **Креативы** — каждому сегменту свой угол, свой ЦТА
5. **Воронка и трафик** — тестирование гипотез

---

## 🔗 ИНТЕГРАЦИЯ

Этот скилл подключается к:
- **Контентная воронка:** Сегменты + уровень осознанности = тип контента
- **Креатив:** Каждый сегмент = свой креатив, свой мессаджинг
- **Трафик:** Горячий сегмент требует меньше потрат, холодный больше
- **LTV:** Разные сегменты = разные LTV

## 🙋 ЧАСТЫЕ ВОПРОСЫ

**Q: Я могу выбрать только 1-2 сегмента?**
A: Можно, но риск выше. Топ-3 с разной теплотой оптимально.

**Q: Может ли сегмент быть "все подряд"?**
A: Нет. Сегмент = узкая группа по ОДНОЙ причине покупки.

**Q: А если мой сегмент не прошёл чек-лист?**
A: Переработай его описание или замени на другой из списка 10-20.

**Q: Когда начинать платить на трафик?**
A: После User Research интервью. Это дает уверенность в гипотезах.

**Q: Могу я делать это в одиночку?**
A: Да. Но с коллегой/наставником проще. Можно обсудить результаты.
