# Фреймворк синтеза — CustDev Report

Для Фазы 5. Сводит интервью + разведку в actionable гипотезы.

```markdown

## TL;DR

{3 строки: стоит / не стоит / стоит с пивотом + почему}

## Карта болей

| Боль | Частота (N из 5) | Интенсивность (1-10) | Готовность платить |
|------|-----------------|---------------------|-------------------|
| {боль 1} | {X/5} | {X} | {да/нет, сумма} |
| {боль 2} | {X/5} | {X} | {да/нет, сумма} |
```

## Критерии оценки жирности

| Оценка | Value Equation | Условия |
|--------|---------------|---------|
| **Grand Slam** | ≥ 30/40 | Боль частая (3+/5), интенсивная (7+), готовы платить, Gap есть |
| **Доработка** | 20-29/40 | Есть потенциал, но нужен пивот — другой формат, цена, или механизм |
| **Слабая** | < 20/40 | Боль слабая, рынок забит, персоны скептичны, нет Gap |
| **Отказ** | любой | 4+ из 5 персон сказали "не нужно", или рынок мёртв |

## Value Equation — как скорить

По Hormozi ($100M Offers):

- **Dream Outcome** — насколько желаемый результат совпадает с тем, что хочет аватар
  - 10: "это именно то, о чём я мечтаю"
  - 5: "было бы неплохо, но не горит"
  - 1: "мне это не нужно"

- **Perceived Likelihood** — насколько верят что получат результат
  - 10: "вижу доказательства, верю в механизм"
  - 5: "может сработает, а может нет"
  - 1: "очередной развод"

- **Time Delay** — как быстро получат результат (инвертированная шкала)
  - 10: мгновенно или за дни
  - 5: недели
  - 1: месяцы или "когда-нибудь"

- **Effort & Sacrifice** — сколько усилий потребуется (инвертированная шкала)
  - 10: "ничего не надо делать" / минимальные усилия
  - 5: умеренные усилия
  - 1: "надо перестроить всю жизнь"

## Что делать после

| Вердикт | Действие |
|---------|----------|
| Grand Slam | → `/hormozi --build` — строить оффер |
| Доработка | → обсудить пивот с Никитой, возможно повторный `/custdev` |
| Слабая | → записать инсайты, архивировать |
| Отказ | → не тратить время, переключиться на другую идею |
