# Шаблон карточки персоны (OCEAN)

Универсальный шаблон. Работает для любой ниши и ЦА.
Методология: Module 1 Project X (сегментация 3W → БПСВ → OCEAN → интервью).

## Два режима

| Режим | Источник данных | Когда |
|-------|----------------|-------|
| **Универсальный** (по умолчанию) | Генерация на основе ниши, рынка, здравого смысла + WebSearch | Любая идея |
| **На базе анкет** (флаг `--наши`) | `knowledge/audience/deep-audience-analysis.md` (350+ анкет) | Идея для текущей ЦА Никиты |

## Инструкции для Persona Builder

### Универсальный режим (по умолчанию)

1. **Определи ЦА из карточки идеи** — кто эти люди? Ниша, профессия, ситуация.
2. **Сегментируй** — используй сегменты из Фазы 1 (горячий/тёплый/холодный).
3. **Создай 5 персон** с распределением:
   - 3 горячий сегмент (ядро — те кто уже ищет решение)
   - 1 тёплый (осознаёт проблему, не ищет активно)
   - 1 холодный (не осознаёт проблему)
4. **OCEAN обязателен.** Каждая персона — уникальная OCEAN-комбинация.
5. **Матрица разнообразия** (quality gate):
   - Возраст: разброс ≥ 15 лет
   - Доход: разброс ≥ 3x
   - OCEAN: минимум 1 персона < 2 и 1 > 4 по каждому параметру
   - Типы: есть фанаты + скептики + случайный
   - Голос: минимум 3 разных стиля речи
6. **Конкретика.** "Женщина 30-40 лет" — не персона. "Алина, 34, Екатеринбург, инстаграм-магазин" — персона.
7. **Сохрани** в `knowledge/custdev/personas/{slug}.md` и обнови INDEX.md.

### Режим "на базе анкет" (--наши)

1. Читай `~/workspace/knowledge/audience/deep-audience-analysis.md`
2. 3 аватара: Застрявший деятель / Выгоревший амбициозный / Ищущий вектор
3. Каждая фраза = дословная цитата или синтез из 3+ реальных ответов
4. OCEAN всё равно добавляй — определяй из характера анкет

## OCEAN — что это (кратко)

| Параметр | Низкий (1-2) | Высокий (4-5) |
|----------|-------------|---------------|
| **O** — Openness | Традиционалист, "работает — не трогай" | Экспериментатор, первым пробует новое |
| **C** — Conscientiousness | Хаотичный, по настроению | Перфекционист, всё в таблицах |
| **E** — Extraversion | Интроверт, минимум слов | Экстраверт, думает вслух |
| **A** — Agreeableness | Антагонист, спорит | Уступчивый, соглашается (ловушка!) |
| **N** — Neuroticism | Спокойный как скала | Тревожный, катастрофизирует |

**КРИТИЧЕСКИ ВАЖНО:** OCEAN определяет КАК персона отвечает, а не ЧТО она отвечает.
Agreeableness 5/5 = ловушка: соглашается со всем, данные ненадёжны. Учитывай при синтезе.
