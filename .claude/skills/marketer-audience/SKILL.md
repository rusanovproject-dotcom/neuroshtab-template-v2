# `/marketer-audience` — Глубокая распаковка ЦА

Методология — `knowledge/audience-framework.md` (БПСВ) + `knowledge/schwartz-awareness.md` (уровни) + `knowledge/classics-compact.md` (JTBD).

## Пререквизит
Должен быть `projects/<main>/ICP.md` с выбранным сегментом. Если нет — запусти `/marketer-revision`.

## Шаги

### 1. Прочитай контекст
- `projects/<main>/ICP.md` — кого распаковываем
- `projects/<main>/customers.md` — все дословные цитаты из реальных встреч
- `projects/<main>/audience.md` — если уже есть черновик (дополняем, не переписываем)
- `knowledge/audience-framework.md` секция БПСВ — методология

### 2. Voice of Customer Mining (минимум 20 цитат)
Источники по приоритету:
1. `customers.md` — дословные цитаты покупателей (самый ценный источник)
2. Транскрипты в `transcripts/` — прямая речь
3. Если данных мало — идёшь в рынок: Reddit / форумы / отзывы / комментарии конкурентов

### 3. Собери БПСВ
- **Боли** — минимум 5 (формула «болит / не хватает»)
- **Потребности** — минимум 5 (зеркало болей — «хочу»)
- **Страхи** делятся на:
  - «Ты как продукт» — минимум 3
  - «Я как потребитель» — минимум 3
- **Возражения** по 3 типам:
  - Ценовые (минимум 2)
  - Ценностные (минимум 2)
  - Доверительные (минимум 2)

### 4. 4 уровня осознанности (Schwartz)
Для сегмента распредели типичные тезисы по уровням: Indifference → Awareness → Consideration → Choice. Определи **откуда тянем** и **куда тянем** в воронке.

### 5. JTBD-формула (Christensen)
*«Когда [ситуация], я хочу [мотивация], чтобы [результат]»* — должна пройти тест подстановки (замени имя/нишу — ломается? значит специфично).

### 6. Anti-Avatar
Кого НЕ берём + почему + как отсекаем на первом касании.

### 7. Запиши в `projects/<main>/audience.md`
Append-only. Если файл существовал — новая секция `## YYYY-MM-DD — углубление` с инкрементом. Старое не трогай.

### 8. Обнови `client-profile.md`
Секция «Маркетинговый срез» → обновить `Main Pain`, `Dream Outcome`, `Awareness Level`.

## Quality Gate
- [ ] 20+ дословных цитат Voice of Customer
- [ ] БПСВ заполнен полностью (все 4 блока + подтипы)
- [ ] 4 уровня осознанности + точка входа
- [ ] JTBD прошёл тест подстановки
- [ ] Anti-Avatar с критерием отсечения
- [ ] audience.md обновлён append-only
- [ ] client-profile.md синхронизирован

## Выход
3-5 находок клиенту + топ-5 сильных цитат для копирайтера + переход: *«Глубина собрана. Хочешь продуктовую лестницу под этот сегмент (/marketer-ladder)?»*
