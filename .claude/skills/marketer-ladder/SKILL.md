# `/marketer-ladder` — Продуктовая лестница

Методология — `knowledge/audience-framework.md` секция «Продуктовая матрица» + `knowledge/hormozi-offer-market.md` (Value Equation, Lead Magnet critera).

## Пререквизит
Должны существовать `projects/<main>/ICP.md` + `audience.md`. Если нет — сначала `/marketer-revision` и `/marketer-audience`.

## Шаги

### 1. Контекст
- `ICP.md` — кто клиент, Dream Outcome, Awareness Level
- `audience.md` — боли, возражения (для логики ступеней)
- `customers.md` секция «Что работает» — реальные паттерны покупок
- `hypotheses.md` активные `[PRODUCT]`, `[PRICE]`, `[LM]`, `[OFFER]` гипотезы

### 2. Собери 5 ступеней (по у)
```
[0] Lead Magnet      — узкая проблема решена полностью (по Хормози)
[1] Tripwire         — дешёвый вход, окупает CAC (100-990 ₽)
[2] Core Offer       — основной продукт (зарабатываем здесь)
[3] Profit Maximizer — upsell / cross-sell / recurring
[4] Return Path      — возврат клиента + рефералы
```

Для каждой ступени:
- **Что это конкретно** (не «курс», а «7-дневный аудит с персональным отчётом»)
- **Цена**
- **Какую боль решает** (из audience.md)
- **Какую проблему создаёт** (для перехода на следующую ступень)
- **Конверсия ожидаемая** (гипотеза из customers паттернов)

### 3. Логика связок
Lead Magnet создаёт проблему которую решает Tripwire. Tripwire создаёт проблему которую решает Core. И т.д. Если цепочка ломается — пересобирай.

### 4. Value Equation для Core Offer (Хормози)
Числитель (увеличить): Dream Outcome + Perceived Likelihood
Знаменатель (уменьшить): Time Delay + Effort & Sacrifice

### 5. Starving Crowd тест для Core
4 критерия × 10 баллов. Меньше 25/40 — продукт не полетит, пересобирай.

### 6. Запиши в `projects/<main>/product-ladder.md`
Структура: краткая карта лестницы + детали каждой ступени + конверсии + гипотезы.

### 7. Поставь H-гипотезы в `hypotheses.md`
Минимум 2 новые гипотезы с тегами:
- `[PRODUCT]` — «ступень X работает для сегмента Y»
- `[PRICE]` — «цена Tripwire N работает лучше чем M»

## Quality Gate
- [ ] Все 5 ступеней описаны конкретно (не «курс», а что внутри)
- [ ] Цены проставлены
- [ ] Связки между ступенями логичны
- [ ] Core прошёл Value Equation check
- [ ] Starving Crowd тест ≥ 25/40
- [ ] 2+ новые гипотезы в hypotheses.md

## Выход
Карта лестницы + *«Core готов. Следующий шаг — воронка под эту лестницу (/marketer-funnel) или углубление оффера?»*
