# Примеры распределения по уровням осознанности — Good / Bad path

> Слой ПРИМЕРЫ: реальные кейсы из Урока 4

## Good Path: Инвестирование (распределение тезисов Олега)

### Контекст
- Продукт: курс по инвестированию
- Тема: финансовая грамотность / инвестирование
- Респондент: Олег, горячий сегмент — предприниматель с капиталом

### Уровень 1 — Indifference
```
(у Олега тезисов на этом уровне НЕТ — он осведомлён)

Пример от другого респондента:
1. "Да я государственник пробитый, мне это не нужно"
2. "Не думал об этом никогда, и так нормально всё"
3. "Какие-то форексы, что-то слышал, ха-ха"
```

### Уровень 2 — Awareness
```
1. "Знаю что есть облигации, акции, IPO, дивидендная стратегия"
2. "Слышал про финграмотность, но не погружался"
3. "У знакомого хорошо получается с инвестициями"
```

### Уровень 3 — Consideration
```
1. "Мне это нужно чтобы припарковать капитал"
2. "Достаточно ли я доверяю этой системе?"
3. "Что будет если я не начну инвестировать — потеряю деньги на инфляции"
4. "Смогу ли я совмещать с бизнесом?"
```

### Уровень 4 — Choice
```
1. "IPO конечно классно, но я думал о блокчейне"
2. "Мне ближе недвижка — низкий риск"
3. "Хочу быть пассивным инвестором, венчур не для меня"
4. "Почему именно этот курс, а не Кийосаки?"
```

### Вывод
- Точка входа: уровень 3 (Consideration) — уже осведомлён, примеряет на себя
- Точка выхода: уровень 5 (покупка)
- Путь: 3 → 4 → 5 (короткая воронка, 2 уровня)
- Стратегия: мотивация + доказательства + оффер

## Good Path: Кондиционеры (триггер через обстоятельства)

### Уровень 1 → Уровень 3 (перепрыгнул)
```
Было: 22-23 градуса → "Вроде нужно, вроде нет, можно поставить, можно не поставить"
Стало: 37 градусов → сразу побежал искать, примерил на себя, начал выбирать

Инсайт: внешние обстоятельства могут перепрыгнуть уровни.
Жара = триггер, который перевёл из Indifference сразу в Consideration.
```

### Уровень 4 — Choice (кондиционер)
```
Критерий выбора: СКОРОСТЬ монтажа
"Здрасте, ребят, только через 2 недели можем" → "Всё, спасибо, понятно"
→ Звонит следующему. Критерий = не цена, а срок.
```

## Bad Path: типичные ошибки классификации

### Ошибка 1: Путать Awareness и Consideration
```
❌ Тезис "Интересно бы узнать про автоворонки" → Consideration
✅ Это Awareness — человек собирает инфу о ТЕМЕ, не примеряет на себя

❌ Тезис "Мне это нужно для автоматизации запусков" → Awareness
✅ Это Consideration — он уже примеряет ПРОДУКТ на себя, есть мотивация
```

### Ошибка 2: Путать тему и продукт
```
❌ На уровне 1-2 говорить о продукте: "Не знает про наш курс"
✅ На уровне 1-2 говорить о теме: "Не знает про инвестирование вообще"

❌ На уровне 3-4 говорить о теме: "Интересуется маркетингом"
✅ На уровне 3-4 говорить о продукте: "Примеряет курс по автоворонкам на себя"
```

### Ошибка 3: Все тезисы на одном уровне
```
❌ 15 тезисов на Consideration, 0 на остальных
✅ Если так — респондент не репрезентативен, или неправильно классифицировал.
   Перечитай ответы, ищи нюансы: где он был безразличен, где выбирал.
```

### Ошибка 4: Перефразировать тезисы
```
❌ Переписать своими словами: "Респондент заинтересован в теме"
✅ Дословно: "Неплохо было бы, надо почитать что-нибудь по этому поводу"
   Ctrl+C из транскрипта → Ctrl+V в таблицу
```
